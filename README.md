### **Installation**
+ via **CDN**
``` html
  <script src="https://unpkg.com/vue-navigation-plugin/dist/vue-navigation-plugin.umd.min.js"></script>
```

+ via **NPM**
```bash
$ npm install vue-navigation-plugin --save # Or
$ yarn add vue-navigation-plugin
```

#### see example for `/test`

#### Build
```bash
npm run build
```
